# **App Name**: Zenith Commerce

## Core Features:

- Product Catalog: Display a curated catalog of science-backed, sustainable products with detailed descriptions, images, and pricing.
- Shopping Cart: Enable users to add products to a shopping cart and manage quantities before proceeding to checkout.
- Secure Checkout: Integrate a secure payment gateway to process transactions with options for various payment methods.
- Order History: Allow users to view their past orders, track shipments, and manage returns.
- Personalized Recommendations: Use a tool to provide product recommendations based on browsing history, purchase behavior, and user preferences.
- Product Reviews: Display user reviews and ratings for each product to build trust and provide social proof.
- Sustainable Sourcing Badges: Visually highlight products that meet specific sustainability criteria with identifiable badges.

## Style Guidelines:

- Primary color: Neon yellow (#Eaff2A) for main call-to-action elements to convey a modern and friendly feel.
- Background color: Off-white (#f4f4f2) for a calm and minimal foundation.
- Accent color: Warm burnt orange (#e36b3d) used sparingly for emotional emphasis, particularly in review sections.
- Font: 'Inter' sans-serif as the primary font for its neutral, modern, and highly readable qualities. Note: currently only Google Fonts are supported.
- Font weights: Strict adherence to font weights (400, 500, 600, 700) to maintain consistency and avoid overuse of bold.
- Spacing: Use of an 8px grid system with predefined spacing tokens to ensure consistent spacing throughout the site.
- Simple iconography with consistent stroke, using circle checkmarks to reinforce trustworthiness.
